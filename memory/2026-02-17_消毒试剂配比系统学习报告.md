## 一、系统功能理解

### 1.1 系统定位
消毒试剂配比系统是柳州市疾病预防控制中心消毒与病媒生物防制所开发的移动端专业工具，旨在解决一线消毒人员在现场配比计算中的实际困难，确保消毒工作科学、规范、安全。

### 1.2 核心功能模块

| 功能模块 | 功能描述 | 技术特点 |
|---------|---------|---------|
| **配比计算** | 支持"计算稀释倍数"和"计算配制量"两种模式 | 自动单位换算（%与mg/L），智能识别固体/液体剂型 |
| **传染病方案** | 覆盖甲类（2种）、乙类（27种）、丙类（11种）传染病 | 依据GB19193-2025标准，一键应用方案到计算器 |
| **推荐浓度** | 11大消毒场景的标准浓度查询 | 含环境、餐饮具、果蔬、织物、手、皮肤、器械、空气、疫源地、饮用水、污水、冷链 |
| **配比记录** | 自动保存每次计算记录 | 本地存储，支持CSV导出，数据持久化 |
| **常用方案** | 保存个人常用配比方案 | 一键快速调用，提高工作效率 |

### 1.3 消毒剂数据库

系统内置5大类20余种消毒剂：

1. **含氯消毒剂**：84消毒液、漂白粉、次氯酸钠、二氯异氰尿酸钠、三氯异氰尿酸、氯胺T
2. **醇类**：95%乙醇、无水乙醇
3. **过氧化物类**：过氧化氢、过氧乙酸、二氧化氯（稳定型/泡腾片）
4. **季铵盐类**：苯扎溴铵、双链季铵盐
5. **其他**：碘伏、碘酊、来苏儿、戊二醛

### 1.4 技术亮点

- **智能剂型识别**：自动区分固体/液体，提供不同的操作指引（固体按重量g，液体按体积mL）
- **动态单位切换**：含氯消毒剂支持%与mg/L双向换算，二氧化氯专用mg/L模式
- **离线使用**：Android APK和PWA网页版均支持离线使用
- **数据本地化**：记录和方案存储在本地，保护敏感工作数据

   - 说明书较为详细，但缺乏"5分钟快速上手"的简化版指引
