# 党政机关公文排版规范

本技能依据《党政机关公文格式》国家标准（GB/T 9704-2012）制定，用于创建和检查符合规范的党政机关公文。

## 快速使用指南

### 创建新公文

1. 确定公文类型（通知、请示、报告、函等）
2. 使用 `assets/` 目录中的对应模板
3. 按规范设置页面和格式
4. 填写公文各要素内容

### 检查现有公文

使用规范要求逐项检查：
- 页面设置（页边距、纸张）
- 字体字号
- 各要素位置
- 行距和段落

## 公文格式规范

### 页面设置

| 项目 | 要求 |
|------|------|
| 纸张规格 | A4型纸（210mm×297mm） |
| 上白边（天头） | 37mm±1mm |
| 下白边 | 35mm±1mm |
| 左白边（订口） | 28mm±1mm |
| 右白边 | 26mm±1mm |
| 版心尺寸 | 156mm×225mm |
| 每页行数 | 22行 |
| 每行字数 | 28个字 |
| 行距 | 固定值28-29磅 |

### 字体字号规范

| 要素 | 字体 | 字号 | 说明 |
|------|------|------|------|
| 份号 | 黑体 | 6位3号 | 顶格编排在版心左上角 |
| 密级和保密期限 | 黑体 | 3号 | 顶格编排在版心左上角第二行 |
| 紧急程度 | 黑体 | 3号 | 顶格编排在版心左上角第三行 |
| 发文机关标志 | 小标宋体 | 红色 | 居中排布，推荐使用红色小标宋体 |
| 发文字号 | 仿宋_GB2312 | 3号 | 居中排布在发文机关标志下 |
| 签发人 | 仿宋_GB2312 | 3号 | "签发人"三字用仿宋 |
| 签发人姓名 | 楷体_GB2312 | 3号 | 右空1字 |
| 标题 | 方正小标宋简体 | 2号 | 居中，多行排列使用梯形或菱形 |
| 主送机关 | 仿宋_GB2312 | 3号 | 顶格，冒号全角 |
| 正文 | 仿宋_GB2312 | 3号 | 首行缩进2字符 |
| 附件说明 | 仿宋_GB2312 | 3号 | 正文下空1行左空2字 |
| 发文机关署名 | 仿宋_GB2312 | 3号 | 右空4字编排 |
| 成文日期 | 仿宋_GB2312 | 3号 | 用阿拉伯数字标全 |
| 印章 | 端正、居中 | - | 下压发文机关署名和成文日期 |
| 附注 | 仿宋_GB2312 | 3号 | 左空2字加圆括号 |
| 附件 | 仿宋_GB2312 | 3号 | "附件"二字及顺序号用黑体 |
| 抄送机关 | 仿宋_GB2312 | 4号 | 左右各空1字 |
| 印发机关和日期 | 仿宋_GB2312 | 4号 | 末条分隔线之上 |
| 页码 | 宋体 | 4号半角 | 单页居右，双页居左 |

### 公文要素编排顺序

```
版头部分（红色分隔线以上）：
1. 份号
2. 密级和保密期限
3. 紧急程度
4. 发文机关标志
5. 发文字号
6. 签发人（上行文必备）
7. 版头中的分隔线

主体部分：
8. 标题
9. 主送机关
10. 正文
11. 附件说明
12. 发文机关署名、成文日期和印章
13. 附注
14. 附件

版记部分（黑色分隔线以下）：
15. 抄送机关
16. 印发机关和印发日期
17. 版记中的分隔线
18. 页码
```

## Word排版实现

### 页面设置代码

```python
from docx import Document
from docx.shared import Mm, Pt, Inches
from docx.enum.text import WD_ALIGN_PARAGRAPH

# 创建文档
doc = Document()

# 设置页面大小和边距
section = doc.sections[0]
section.page_width = Mm(210)  # A4宽度
section.page_height = Mm(297)  # A4高度
section.top_margin = Mm(37)    # 上白边
section.bottom_margin = Mm(35) # 下白边
section.left_margin = Mm(28)   # 左白边（订口）
section.right_margin = Mm(26)  # 右白边
```

### 设置行距和字体

```python
from docx.shared import Pt

# 设置正文段落格式
def set_body_paragraph_format(paragraph):
    paragraph.paragraph_format.line_spacing = Pt(28.95)  # 固定值约29磅
    paragraph.paragraph_format.space_after = Pt(0)
    paragraph.paragraph_format.space_before = Pt(0)
    
    # 设置字体
    for run in paragraph.runs:
        run.font.name = '仿宋_GB2312'
        run.font.size = Pt(16)  # 3号字 = 16磅
        run._element.rPr.rFonts.set(qn('w:eastAsia'), '仿宋_GB2312')
```

## 常见公文类型模板

模板文件存放在 `assets/` 目录：
- `assets/通知模板.docx` - 下行文（向下级机关行文）
- `assets/请示模板.docx` - 上行文（向上级机关请示）
- `assets/报告模板.docx` - 上行文（向上级汇报工作）
- `assets/函模板.docx` - 平行文（不相隶属机关间行文）

- [ ] 行距固定值28-29磅

- [ ] 首行缩进2字符

- [ ] 成文日期用阿拉伯数字
- [ ] 印章端正、居中
